# Walrus Storage Costs Calculation Utilities and Resources
In this document you will find useful resources and utilities for Storage Costs calculation and more on the Walrus Protocol

## Storage Cost Calculators 

### Cost Calculator (official)
A Storage Cost Calculator by Walrus Team can be found at:
https://costcalculator.wal.app/

### Cost Calculator (by Rubynodes)
Another unofficial Cost Calculator can be found at:
https://walrus-cost-calculator.rubynodes.io/

## Guides and Docs

### Detailed guide about Walrus Storage Costs from Walrus Docs
This useful guide by Walrus/ML team provides more details about storage costs in Walrus:
https://docs.wal.app/docs/system-overview/storage-costs

### Walrus Blob IDs and Sui Object IDs relation from Walrus Docs
This document from Walrus/ML team provides details about the relation between Walrus Blob IDs and their counterpart/representation in Sui as Object IDs: 
https://docs.wal.app/docs/system-overview/sui-object-and-blob-ids

### Walrus SDK from MystenLabs
The Walrus SDK documentation from ML provides details about available methods for the Typescript SDK, including simulating transactions in Sui for final total cost(WAL+SUI):
https://sdk.mystenlabs.com/walrus#writing-files

## Explorers

### Walrus Explorer
An explorer/scanner for Walrus, which provides real time data about the network, such as:
- Total Epochs
- Total Blobs
- Current storage prices
- ...and much more  

can be found at:
https://walruscan.com
